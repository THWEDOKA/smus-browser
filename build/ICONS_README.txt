Поместите иконки браузера в эту папку для сборки:

Windows:
  icon.ico - иконка для Windows (256x256, .ico формат)

Linux:
  icon.png - иконка для Linux (512x512, .png формат)

macOS:
  icon.icns - иконка для macOS (.icns формат)

Вы можете создать иконки из логотипа SMUS используя онлайн конвертеры:
- https://www.icoconverter.com/ (для .ico)
- https://cloudconvert.com/png-to-icns (для .icns)

Если иконки отсутствуют, electron-builder использует дефолтные иконки Electron.
